森野小屋 v4｜情侣对话升级版

本版本在 v3“图片与备份升级版”的基础上加入情侣对话区，v3 的多图、3:5 展示、排序、大图查看和图片备份功能均保留。

一、本阶段新增
1. “老公来信”页面升级为“我们的对话”。
2. 青青使用“小西瓜”头像，消息显示在右侧，气泡为青苹果色。
3. 老公使用“小花花”头像，消息显示在左侧，气泡为柠檬黄色。
4. 可切换发送人，记录青青和老公双方说过的话。
5. 自动记录消息时间；修改后显示“已修改”。
6. 支持单条消息编辑、删除，以及清空全部对话。
7. Ctrl + Enter（Mac 为 Command + Enter）可快速发送。
8. 原“抽一封老公来信”功能保留，并可一键把来信以老公身份发进对话。
9. 情侣对话保存在浏览器本地，关闭网页后仍保留。
10. 完整 JSON 备份现同时包含日记、全部图片及情侣对话；恢复时校验发送方和消息顺序。
11. 纯文字导出也会在日记后附上青青与老公的对话。

二、使用方法
1. 解压整个文件夹。
2. 双击 index.html 打开。
3. 点击底部“我们的对话”。
4. 选择“青青”或“老公”，输入文字后发送。
5. 需要备份时，进入“回忆”，点击“导出完整备份”。

三、重要提醒
- 上传 GitHub 时，请上传解压后的全部文件和文件夹，不要只上传 index.html。
- avatars、css、js 文件夹缺失会导致头像、样式或功能失效。
- 数据仍只保存在当前浏览器中；云端保存、多设备同步和自动备份将在后续阶段加入。
- 导入旧版 v2/v3 日记备份时，旧日记和图片仍可正常恢复；旧备份没有情侣对话属于正常情况。

四、主要新增文件
- assets/avatars/qingqing-watermelon.jpeg：青青的小西瓜头像
- assets/avatars/laogong-flowers.jpeg：老公的小花花头像
- js/chat.js：情侣对话的数据校验、本地保存、合并和恢复校验
