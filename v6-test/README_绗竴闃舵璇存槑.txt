森野小屋 v3｜第一阶段：图片与备份升级

本版新增：
1. 日记图片统一使用 3:5 画幅显示。
2. 支持一次选择多张图片。
3. 支持拖动排序，以及“前移 / 后移”按钮排序。
4. 支持逐张删除与清空当前图片。
5. 支持点击图片查看大图、上一张、下一张和键盘方向键切换。
6. 旧版单张 photo 数据会自动迁移为新版 photos 数组。
7. JSON 完整备份包含全部图片及顺序。
8. 导出前提示日记数、图片数和预计大小。
9. 导入前提示备份内容；导入后核对恢复前后日记数和图片数。
10. 恢复完成后自动校验日记、图片位置与顺序。
11. 项目已拆分为 HTML、CSS、图片处理、日记数据、备份和主程序文件，方便后续继续升级。

使用方法：
- 解压整个文件夹。
- 双击 index.html 即可本地打开。
- 上传 GitHub Pages 时，请上传整个文件夹内的全部文件和子文件夹。

建议测试：
- 新建一篇日记，上传 3-5 张照片并排序。
- 保存后重新编辑，检查顺序。
- 点击时光轴图片检查大图切换。
- 导出 JSON 完整备份，清空测试数据后再导入恢复。

注意：
- 当前仍使用浏览器 localStorage，本阶段尚未加入登录和云端。
- 多图会更快占用浏览器空间，请定期导出完整备份。
