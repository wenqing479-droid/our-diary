森野小屋 v5｜青苹果与绿色小外星人美化版

本版基于 v4.1 制作，原有功能全部保留：
- 3:5 多图日记、排序、删除与大图查看
- 情侣对话、双方气泡、可自由切换情头
- 日记、图片、对话和头像完整备份与恢复

本次新增：
1. 24 张绿色小外星人素材全部放入 assets/aliens/，已优化为 WebP，减少网页体积。
2. 从青苹果素材母版中挑选 6 个无水印区域，制作成透明背景 PNG，放入 assets/apples/。
3. 青苹果加入网页标题、面板标题、表单标签、保存按钮、照片上传区、提示气泡和小屋门牌。
4. 写日记、时光轴、情侣对话、回忆和备份页面加入不同的小外星人装饰。
5. 时光轴空白页、聊天空白页和照片空白页增加专属素材提示。
6. 回忆页面新增“今天是哪只小外星人”，每天默认出现一只，也可手动随机切换，24只都会轮流出现。
7. 日间与夜间模式、手机端和减少动画偏好均已适配。

使用方法：
- 解压后直接打开最外层 index.html。
- 上传 GitHub Pages 时，需要把 index.html、css、js、assets 和说明文件一起上传。
- assets 文件夹不可遗漏，否则青苹果、头像和小外星人不会显示。

版本说明：
- 本次只调整静态素材和视觉表现，不改动日记、聊天、头像或备份数据结构。
- 旧版浏览器中已有的数据会自动继续读取。
