森野小屋 v4.1｜头像自由切换版

一、新增功能
1. 在“我们的对话”顶部点击青青头像、老公头像，或点击“更换情头”，即可展开头像设置。
2. 青青和老公可以分别上传任意图片作为头像。
3. 图片会从中央自动裁成正方形，并压缩为 640×640 的 JPEG，聊天中显示为圆形。
4. 更换后，顶部情头、发送身份按钮、历史对话头像及“老公来信”头像会立即同步更新。
5. 关闭网页再打开，自定义头像仍然保留。
6. 可随时恢复默认头像：青青恢复“小西瓜”，老公恢复“小花花”。
7. 自定义头像已经加入完整 JSON 备份，恢复后会跟随日记、照片和对话一起回来。
8. 兼容 v1—v4 旧备份；旧备份没有头像数据时，不会破坏当前头像。

二、使用方法
1. 解压整个压缩包。
2. 双击最外层 index.html。
3. 点击底部“我们的对话”。
4. 点击“更换情头”，分别上传青青或老公的新头像。
5. 单张头像文件不要超过 12MB。

三、注意事项
- 日记、照片、情侣对话与自定义头像目前仍保存在当前浏览器。
- 换设备、清理浏览器数据或重装浏览器前，请先到“回忆—备份与迁移”导出完整备份。
- 上传 GitHub 时，请上传解压后的全部文件和文件夹，不能只上传 index.html。
